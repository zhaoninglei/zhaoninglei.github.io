## 2. 机器人启动教程

1. **右旋机器人底盘后侧旋钮至水平**，等待机器人程序启动。

2. **观察机器人底盘前方灯带**：
   - 灯带会由**长蓝**转为**黄色**，表示底盘 ROS 服务启动成功。

3. **语音服务启动判断**：
   - 若机器人已配置网络，开机约 **1 分钟** 内会播报“语音服务启动成功”。
   - 若未配置网络，则语音功能启动失败。
   - 配置网络后可再次开机测试。网络配置教程详见：[机器人网络连接文档.md](https://mcngj4ls8sie.feishu.cn/wiki/D20PwaJQni6myRkzMTFcQEu4nGd?fromScene=spaceOverview)

4. **机械臂使能确认**：
   - 开机约 **1 分钟** 后，机械臂全部电机使能。
   - 用手轻轻旋转 8 个关节电机，应处于**不可动状态**，表示机械臂功能启动成功。
